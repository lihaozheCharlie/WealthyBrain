# 复盘检查清单

用于单只或批量复盘。

| ID | 问题 |
| --- | --- |
| WB-REVIEW-01 | 原始参考价是否正确且有日期？ |
| WB-REVIEW-02 | 当前价格和复权价格是否可得？ |
| WB-REVIEW-03 | 简单收益率、基准收益率和超额收益率分别是多少？ |
| WB-REVIEW-04 | 原始投资假设是兑现、失败，还是仍未解决？ |
| WB-REVIEW-05 | 结果由市场 beta、行业 beta、个股新闻、估值还是运气驱动？ |
| WB-REVIEW-06 | 是否出现任何失效条件或停止复盘触发？ |
| WB-REVIEW-07 | 哪些检查项预测了结果？ |
| WB-REVIEW-08 | 哪些检查项没能捕捉风险？ |
| WB-REVIEW-09 | 应记录什么经验？ |
| WB-REVIEW-10 | 规则应升级、降级，还是继续观察？ |
